PM工具盘点：那些鲜为人知却好用到哭的效率神器

Aug 19, 2017 | i


有一句老话说得很好：工欲善其事，必先利其器。在飞速发展的互联网时代，善于利用工具可以让我们事半功倍，从而把更多的精力放在提升核心竞争力。笔者平时喜欢搜罗体验产品工具，在此将一些好的工具整理出来推荐给大家。

1.Notion笔记：印象笔记完美替代品？任务、文件、知识库它都能记。


Notion 自称是一款「将笔记、知识库和任务管理无缝整合的协作平台」。它具有无限的层级和相互链接的组织弹性，给笔记间的关系提供了足够自由的组织方式；它排版灵活，能够把笔记里的内容按块（Block）进行组织和拖拽，甚至可以做出 Trello 看板进行项目管理；它的内容类型丰富，可以嵌入图片、网页、文档甚至视频，几乎无所不能。

2.伙伴云表格：需求管理、跟进排期、收集bug的数据利器。


伙伴云表格是一款针对多人数据协作工具，主要用于导入本地excel文件，实现在线数据共享与协作。软件创新的移动端体验，解决Excel文件在移动端只能看，不能增、删、改、查的问题，做到随时随地管理业务数据。

3.MindNode和幕布：你负责脑洞大开，它负责呈现想法。

（1）MindNode

脑图是帮助你展现思维和想法的一种视觉化工具，它基本是由两个部分组成的：节点和联系。

（2）幕布
幕布是一款结合了大纲笔记和思维导图的头脑管理工具,帮你用更高效的方式和更清晰的结构来记录笔记、管理任务、制定计划甚至是组织头脑风暴。

4.Omnigraffle和ProcessOn：无法全览所有的页面就画流程图。

（1）Omnigraffle

OmniGraffle主要用于绘制流程图、图表、组织结构图、UI界面设计等等，你可以用于设计任何描述图形，简单易用，功能强大，被誉为Mac上的Visio，非常的专业和优秀。

（2）ProcessOn

ProcessOn是一个在线作图工具的聚合平台，它可以在线画流程图、思维导图、UI原型图、UML、网络拓扑图、组织结构图等等，您无需担心下载和更新的问题，不管Mac还是Windows，一个浏览器就可以随时随地的发挥创意，规划工作。

5.Balsamiq Mockups：用草图原型快速表达你的想法。


Balsamiq Mockups是一款快速草图原型工具，上手容易，整体呈现手绘风格。比较适合用来表达产品设计概念和基本布局，但不适合做带有交互性质的原型设计。

6.LICEcap：用gif就能轻松表达难以阐述的流程。


LICEcap是一款屏幕录制工具，支持导出 GIF 动画图片格式，轻量级、使用简单，录制过程中可以随意改变录屏范围。

7.Eagle：设计师灵感的收集器。


Eagle 是一款专为设计师打造的图片管理软体，能够帮助你轻松管理电脑上各种图片、照片、荧幕截图、介面设计、网页设计、平面设计、Logo设计、Sketch、汇入Pinterest 等各种图片，并且提供云端同步功能让你可以在多台装置同步收藏的素材，也可以与你的团队协同共用，另外 Mac 与 Windows 系统的使用者，现在都可以下载并使用！

8.Zeplin和蓝湖：从此告别切图和标注！

（1）Zeplin
Zeplin是一款为 Designer 和 Developer 提供产品原稿交接与沟通的高效平台，用户可对产品进行标注、Style Guide 注释，还提供了简单的团队协同工作功能，通过插件，设计师可将 Sketch的 Artboard 单独或批量导入 Zeplin 的 Project 里供开发者做业务功能与前端交互的实现。

（2）蓝湖
蓝湖是一款产品设计的协作平台,帮助互联网团队更高效地进行设计。蓝湖可以为设计图自动生成标注,与团队共享设计图,展示页面之间的跳转关系。

9.chrome插件：设计师不能错过的小插件。

(1)掘金

掘金 Chrome 插件，专为开发者、设计师和产品经理量身定制，能够帮助你在打开新标签页的时候，快速发现优质的开发、设计或者产品方面的干货，减少筛选和操作成本。

(2) Responsive Web Design Tester

只要你安装了Responsive WebDesign Tester，你就可以自由地查看网页上画面的尺寸和分辨率。在智能手机已经普及的现在，对于响应式网站来说这可是必不可少的工具。

(3)What Font

这能在网页上直接使用，让你知道当前网站上使用了哪种字体的工具。如果你还在使用像Firebug 或 Webkit这样的工具，不如来试试What Font吧！只要通过点击就能够很方便的确认网站上使用的字体，是不是非常的方便呢？

(4)Colorzilla

Colorzilla 是可以即时的看出网站上所使用颜色的RGB和HEX值，并且可以随时保存的一个浮动工具栏。他的另一个特长是点击「Webpage Color Analyzer」就可以看到网站的配色。

10.Dropbox和坚果云：所剩无几的云同步网盘

(1)Dropbox

Dropbox是最实用且免费的文件同步、备份、共享云存储软件。目前已经超过一亿用户正在使用Dropbox，如果你也有文件同步、备份、共享的需求，请选择Dropbox，谁用谁知道。

(2)坚果云

坚果云是一款专业、安全的效率类工具、支持任意设备上的文件同步。可以通过它改善自己的工作效率，将真正有价值的文件资料留存下来。

11.纸：最原始的或许是最好的


原型草稿绘图网点本，如果是开发时间紧迫并且逻辑比较简单的产品，完全可以用纸上原型来搞定啊。

12.原型尺：手残党的压箱底神器


原型尺有APP版和WEB版，都是最主流的尺寸，上面的icon也是产品设计中出现频率很高的icon，绘制一个普通的原型，绝对够够的。

13.mac+外接显示器：原来我也可以快乐地搬砖～

笔者选购的是13寸macbook pro定制机，定制机的配置是跟15寸的一样，而且加上外接显示器则拥有比15寸更大的视野，这样子在家工作非常享受。出门携带也十分方便，不像15寸那样笨重。

总结

axure和sketch这些工具早已家喻户晓，所以这次没讲。笔者认为工作职责决定使用工具，如果产品只需要表达自己大概的产品概念，其实也完全用不到axure，可以试试原型草稿绘图网点本。
总的来说：先了解自己，再选择工具。
